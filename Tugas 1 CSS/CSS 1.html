PROFIL UNIVERSITAS GAJAYANA MALANG

Universitas Gajayana Malang, adalah perguruan tinggi swasta di Malang, Indonesia,Universitas ini di bawah Yayasan Pendidikan Gajayana. Berdirinya Universitas Gajayana diawali dengan sebuah Akademi Manajemen Perusahaan dan Akuntasi Malang pada tahun 1980. Empat tahun sesudah itu akademi ini ditingkatkan menjadi Sekolah Tinggi Ilmu Ekonomi Gajayana Malang. Selanjutnya mulai pada tanggal 28 Oktober 1986 statusnya berubah menjadi Universitas Gajayana hingga sekarang. Rektor Universitas Gajayana saat ini adalah: Prof. Dr. Hj. Dyah Sawitri, M.M.

Program studi

Universitas Gajayana Malang memiliki 3 Fakultas, 9 Program Studi dan 2 Program Pascasarjana yang telah Ter-Akreditasi, yaitu:

Fakultas Ekonomi dan Bisnis (FEB)

Program Studi Akuntansi Program Studi Manajemen Program Studi Ekonomi Pembangunan-Perbankan

Fakultas Teknik dan Informatika (FTI)

Program Studi Teknik Elektro Program Studi Teknik Mesin Program Studi Sistem Informasi Program Studi Fisika Program Studi Rekayasa Industri

Fakultas Ilmu Sosial Budaya (FISB)

Program Studi Bahasa dan Sastra Inggris Program Studi Psikologi Program Studi Ilmu Komunikasi Pasca Sarjana
